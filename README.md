**API написаное на Node.js + express.js, которое управляет заметками(Notes):**

Сущность **Note**:
- id
- title (1-100 символов)
- content (1-500 символов)
- createdAt

- Метод **Get** показывает все заметки при обращении по URL: "/notes", одну заметку(по id) при URL: "/notes/:id"
- Метод **Post** создаёт заметку при обращении по URL: "/notes
- Метод **Put** полностью обновляет заметку при обращении по URL: "/notes/:id", нужно обязательно указать title и id менять нельзя, можно указать свою дату создания(если не указано создаётся новая)
- Метод **Patch** частично обновляет заметку при обращении по URL: "/notes/:id", id менять нельзя, можно указать свою дату создания(если не указано остаётся старая)
- Метод **Delete** удаляет заметку при обращении по URL: "/notes/:id"
